Terraform module which deploy a jenkins based on Alibaba Cloud ECS instance.

terraform-alicloud-jenkins
=====================================================================

## Usage

To run this example you need to execute:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

Note that this example will create resources which cost money. Run `terraform destroy` when you don't need these resources.
